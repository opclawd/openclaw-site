Test 4: API GraphQL SpaceX

Consumo de API GraphQL para obtener lanzamientos históricos

Consigna

Realizar un POST a https://spacex-production.up.railway.app/ con body JSON conteniendo una query GraphQL para obtener los últimos 3 lanzamientos, incluyendo nombre de misión, fecha, éxito y nombre del cohete. Guardar la respuesta en exports/spacex_launches.json.

Pasos Técnicos

  1. Crear directorio exports/ para almacenar resultados
  2. Construir query GraphQL con los campos requeridos
  3. Ejecutar curl -X POST con header Content-Type: application/json
  4. Enviar body JSON con la query GraphQL
  5. Guardar respuesta en archivo JSON

Comando Ejecutado

curl -X POST https://spacex-production.up.railway.app/ \
  -H "Content-Type: application/json" \
  -d '{"query":"{launchesPast(limit:3){mission_name launch_date_utc launch_success rocket{rocket_name}}}"}' \
  -o exports/spacex_launches.json
