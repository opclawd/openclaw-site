Image Downloader Gallery

Python requests + HTML gallery with dark theme

Consigna

  • Descargar 10 imágenes desde https://picsum.photos/400/300 usando Python requests
  • Agregar un delay de 2 segundos entre cada descarga
  • Guardar las imágenes como downloads/images/img_001.jpg a img_010.jpg
  • Generar downloads/images/gallery.html con grid layout y dark theme

Pasos Técnicos

Script Python utilizado:

import requests
import time
import os

os.makedirs('downloads/images', exist_ok=True)

for i in range(1, 11):
    url = 'https://picsum.photos/400/300'
    filename = f'downloads/images/img_{i:03d}.jpg'
    
    response = requests.get(url, timeout=30)
    with open(filename, 'wb') as f:
        f.write(response.content)
    
    if i < 10:
        time.sleep(2)

Características de la galería HTML:

  • CSS Grid responsive con auto-fill y minmax(300px, 1fr)
  • Dark theme: background #0a0a0f, cards #14141f
  • Accent colors: gradiente #ff6b35 a #f7931e
  • Hover effects con transform y box-shadow
  • Font Awesome icons
