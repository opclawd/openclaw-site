Gmail Email Automation

Scripts Python para envío automatizado de emails vía SMTP

Consigna

Configurar un sistema de envío de emails automatizado usando Gmail SMTP con smtplib. El script debe soportar envío de emails HTML con adjuntos opcionales.

Pasos Realizados

  • Creación de scripts/send_email.py con smtplib
  • Configuración de conexión SMTP a smtp.gmail.com:587 con STARTTLS
  • Login con credenciales opclawd@gmail.com (App Password)
  • Soporte para emails HTML con codificación UTF-8
  • Soporte para adjuntos múltiples (MIME multipart)
  • Envío de email de prueba con reporte de proyectos (Batch 1)
  • Verificación de entrega exitosa

Código Principal

def send_email(to_email, subject, body_html, attachments=None):
    msg = MIMEMultipart("alternative")
    msg["Subject"] = subject
    msg["From"] = f"OpenClaw Bot <{SMTP_USER}>"
    msg["To"] = to_email
    
    # Cuerpo HTML
    html_part = MIMEText(body_html, "html", "utf-8")
    msg.attach(html_part)
    
    # Archivos adjuntos
    for filepath in attachments or []:
        part = MIMEBase("application", "octet-stream")
        with open(filepath, "rb") as f:
            part.set_payload(f.read())
        encoders.encode_base64(part)
        part.add_header(
            "Content-Disposition",
            f"attachment; filename=\"{filename}\"",
        )
        msg.attach(part)
    
    # Conexión TLS y envío
    with smtplib.SMTP(SMTP_HOST, SMTP_PORT) as server:
        server.starttls(context=ssl.create_default_context())
        server.login(SMTP_USER, SMTP_PASS)
        server.sendmail(SMTP_USER, [to_email], msg.as_string())

Uso del Script

# Email simple
python scripts/send_email.py \\
    --to "destino@email.com" \\
    --subject "Hola" \\
    --body "<h1>Hola Mundo</h1>"

# Email con adjuntos
python scripts/send_email.py \\
    --to "destino@email.com" \\
    --subject "Reporte" \\
    --body "<h1>Adjunto</h1>" \\
    --attach archivo.pdf imagen.png
