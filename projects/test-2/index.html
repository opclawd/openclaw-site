Test 2: API de Tipos de Cambio

Consumo de API REST y procesamiento de JSON

Consigna

Consultar la API pública de tipos de cambio en https://open.er-api.com/v6/latest/USD, extraer las tasas para EUR, GBP, ARS y JPY, y guardar los resultados en un archivo JSON con timestamp.

Pasos Técnicos

  • Realicé una petición HTTP GET usando curl a la API de Exchange Rate
  • La API devolvió un objeto JSON con todas las tasas de cambio basadas en USD
  • Extraje las tasas específicas solicitadas: EUR, GBP, ARS, JPY
  • Generé el archivo exports/exchange_rates.json con los datos filtrados y metadata
