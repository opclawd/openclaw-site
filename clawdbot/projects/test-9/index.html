TEST #9

Monitor de Precios Web

Consigna

Crear y ejecutar scripts/price_monitor.sh que:

  • Use curl para descargar https://books.toscrape.com/
  • Extraiga el título y precio del primer libro usando grep/sed
  • Guarde los datos en logs/price_history.csv (timestamp, title, price)

Pasos Técnicos

1. Creé el script Bash con:

  • Curl para obtener el HTML de la página
  • Grep con expresiones regulares para extraer el título del primer libro
  • Grep para extraer el precio de <p class="price_color">
  • Sed para limpiar y formatear los datos
  • Validación de existencia del archivo CSV con header automático

2. Código del script:

#!/bin/bash
URL="https://books.toscrape.com/"
LOG_FILE="logs/price_history.csv"

# Fetch the page
HTML=$(curl -s "$URL")

# Extract title and price
TITLE=$(echo "$HTML" | grep -oP '<h3>.*?</h3>' | head -1 | \
        grep -oP 'title="[^"]+"' | cut -d'"' -f2)
PRICE=$(echo "$HTML" | grep -oP '<p class="price_color">[^<]+</p>' | head -1 | \
        grep -oP '>[^<]+<' | tr -d '<>' | sed 's/£//g')

# Save with timestamp
echo "$(date '+%Y-%m-%d %H:%M:%S'),$TITLE,$PRICE" >> "$LOG_FILE"

Ejecución
