Web Scraping & Chart.js Visualization

Consigna

Realizar scraping de datos desde https://www.scrapethissite.com/pages/simple/ para obtener información de países y sus poblaciones.

  • Parsear países y población (divs class="country-info", spans class="country-population")
  • Obtener el Top 10 países con mayor población
  • Generar visualización con Chart.js CDN
  • Gráfico de barras horizontal
  • Tema oscuro: fondo #0a0a0f, barras #ff6b35

Tecnologías Utilizadas

Python 3
urllib.request
Chart.js (CDN)
HTML5 + CSS3

Pasos Técnicos

1. Extracción de datos

# Fetch HTML content from scrapethissite.com
import urllib.request

url = "https://www.scrapethissite.com/pages/simple/"
req = urllib.request.Request(url, headers={'User-Agent': 'Mozilla/5.0'})
with urllib.request.urlopen(req) as response:
    html = response.read().decode('utf-8')

2. Parsing de países y población

# Parse countries from extracted markdown content
countries = []
for line in content.split('\\n'):
    parts = line.rsplit(' - ', 1)
    if len(parts) == 2:
        name, pop_str = parts
        pop = int(pop_str.replace(',', ''))
        countries.append((name.strip(), pop))

3. Ordenar y obtener Top 10

# Sort by population descending and get top 10
top10 = sorted(countries, key=lambda x: x[1], reverse=True)[:10]

4. Generar visualización con Chart.js
