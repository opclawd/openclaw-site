GitHub Repository Automation & Code Push

Consigna

Automatizar la creación de repositorios en GitHub y el push de código local mediante:

  • GitHub API (curl) para crear repos remotos
  • Git local para inicializar, commitear y pushear
  • Archivos de configuración (.gitignore, README, LICENSE)
  • Notificación vía Telegram con los links resultantes

Pasos Ejecutados

1

Crear Repositorios via GitHub API

POST a /user/repos con nombre, descripción y auto_init=false

curl -X POST https://api.github.com/user/repos \
  -H "Authorization: token ***" \
  -d '{"name":"openclaw-todo-app",...}'
2

Setup Local y Push

git init, crear .gitignore, README.md, LICENSE MIT, commit y push

git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/.../repo.git
git push -u origin main
3

Repetir para Todos los Proyectos

3 repositorios creados: todo-app, api-tasks, scripts

4

Notificación Telegram

Envío de links a los 3 repos creados exitosamente
