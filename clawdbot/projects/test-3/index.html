Test 3: Web Scraping de Hockey

Extracción de datos tabulares desde HTML

Consigna

Descargar el contenido de https://www.scrapethissite.com/pages/forms/?page_num=5, parsear la tabla HTML de equipos de hockey, y guardar los datos extraídos en un archivo de texto.

https://www.scrapethissite.com/pages/forms/?page_num=5

Pasos Técnicos

  • 1. Descarga: Se utilizó el módulo requests de Python para hacer una petición HTTP GET a la URL objetivo con headers de User-Agent.
  • 2. Parsing: Se implementó un parser HTML personalizado usando html.parser.HTMLParser de la librería estándar (sin dependencias externas).
  • 3. Extracción: El parser identificó la tabla con clase table y extrajo todas las filas (tr) y celdas (td).
  • 4. Procesamiento: Se mapearon los campos: Team Name, Year, Wins, Losses, OT Losses, Win %, Goals For (GF), Goals Against (GA).
  • 5. Persistencia: Los datos se guardaron formateados en exports/hockey_page5.txt.

Resultado

Se extrajeron 25 equipos de hockey (1 header + 25 datos). Los años cubiertos son 1994 y 1995.

Archivo generado: exports/hockey_page5.txt

Vista previa del código utilizado:

import requests
from html.parser import HTMLParser

# Descargar página
url = "https://www.scrapethissite.com/pages/forms/?page_num=5"
response = requests.get(url, headers={"User-Agent": "..."})

# Parsear tabla HTML
class TableParser(HTMLParser):
    def handle_starttag(self, tag, attrs):
        if tag == 'table' and dict(attrs).get('class') == 'table':
            self.in_table = True
        # ... extracción de filas y celdas

parser = TableParser()
parser.feed(response.text)

# Guardar resultados
with open('exports/hockey_page5.txt', 'w') as f:
    for row in parser.rows:
        f.write(f"{row}\n")
